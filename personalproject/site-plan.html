Overview

Purpose

The purpose of The Music Mogul website is to provide fun, beginner-friendly piano education for kids, teens, and homeschool families. The site helps new musicians learn basic piano concepts through interactive tools, visuals, and simple lessons.

Audience

This site is designed for beginners of all ages—especially young students, homeschoolers, and parents looking for an easy way to introduce music learning at home.

Dynamic Elements

  • Practice tracker using DOM updates
  • Lesson filtering with arrays
  • Random practice tips using modules

Wireframes

Home Page Wireframe

The home page wireframe includes a centered hero section featuring a title, a short introductory paragraph, and a main call-to-action button. Below the hero, three featured content boxes highlight key areas of the site such as beginner lessons, practice tools, and learning tips. Lower on the page is a two-column layout: the left side contains an overview of lesson categories, while the right side includes an image or promotional content. The footer appears at the bottom with standard navigation.

Lessons Page Wireframe

The lessons page wireframe begins with a large page title and an introductory description. Below this is a structured list of beginner lesson items, each represented by simple horizontal line shapes in the wireframe. A practice tip box sits beneath the lesson list, including a button that will generate random learning tips in the final site. The layout is intentionally simple and clean so students can focus on the lesson flow without distractions.
