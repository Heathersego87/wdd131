Overview

Purpose

The purpose of The Music Mogul website is to provide fun, beginner-friendly piano education for kids, teens, and homeschool families. The site helps new musicians learn basic piano concepts through interactive tools, visuals, and simple lessons.

Audience

This site is designed for beginners of all ages—especially young students, homeschoolers, and parents looking for an easy way to introduce music learning at home.

Dynamic Elements

  • Practice tracker using DOM updates
  • Lesson filtering with arrays
  • Random practice tips using modules

Content

Home Page

The Home page introduces visitors to The Music Mogul and sets the tone for a fun, colorful learning experience. It includes a hero section with a welcoming headline, supporting text, and a call-to-action button that directs users to the lessons page. Below the hero, visitors will find four feature boxes that highlight the main benefits of the program, such as beginner-friendly lessons, AI-enhanced learning tools, progress tracking, and free starter sheet music. The Home page also includes an interactive Practice Tracker where users can log practice time. This tracker updates live using JavaScript and helps new learners stay motivated. The footer contains attribution and site navigation.

Beginner Lessons Page

The Beginner Lessons page provides a structured starting point for new piano students. It begins with an introduction that explains the purpose and flow of the lessons, along with a large header image meant to visually engage younger or beginner visitors. The lessons section lists essential foundational skills such as meeting the keyboard, finger numbers, middle C, rhythm basics, and simple melodies. A dynamic filtering feature allows users to show only “easy” lessons using JavaScript array methods. Beneath the lesson list, users can access an interactive tip generator that displays randomized practice suggestions to improve technique and build confidence.

Additional Tools & Features (Within Pages)

The site includes multiple interactive components to support different learning styles. The Practice Tracker (located on the Home page) encourages daily engagement by allowing users to add practice minutes or reset their progress. The Lesson Filter (located on the Lessons page) uses an array of lesson objects to display only beginner-level content. The Practice Tip Generator provides quick motivation and helpful reminders using JavaScript randomness to display a new tip each time the user clicks the button. These elements help make the music learning process enjoyable, consistent, and interactive for students of all ages.

Wireframes

Home Page Wireframe

The home page wireframe includes a centered hero section featuring a title, a short introductory paragraph, and a main call-to-action button. Below the hero, three featured content boxes highlight key areas of the site such as beginner lessons, practice tools, and learning tips. Lower on the page is a two-column layout: the left side contains an overview of lesson categories, while the right side includes an image or promotional content. The footer appears at the bottom with standard navigation.

Lessons Page Wireframe

The lessons page wireframe begins with a large page title and an introductory description. Below this is a structured list of beginner lesson items, each represented by simple horizontal line shapes in the wireframe. A practice tip box sits beneath the lesson list, including a button that will generate random learning tips in the final site. The layout is intentionally simple and clean so students can focus on the lesson flow without distractions.
